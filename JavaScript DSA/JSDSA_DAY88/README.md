## Roman Numeral to Integer Value

### Challenge
Given a roman numeral, the task is to find its corresponding integer value
Note roman numerals are represented by seven different symbols:
I, V, X, L, C, D, M

The integer values for the same are as follows:
I: 1
V: 5
X: 10
L: 50 
C: 100
D: 500
M: 1000

Note:
* I can be placed before V (5) and X (10) to make 4 and 9. 
* X can be placed before L (50) and C (100) to make 40 and 90. 
* C can be placed before D (500) and M (1000) to make 400 and 900.

Sample Case:

“III” 3

“IV” 4

“IX” 9

“MCMXCIV” 1994
