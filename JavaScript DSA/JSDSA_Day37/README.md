### Reverse a singly linked list

For this challenge, write a code to reverse a singly linked list. To make things more interesting, try doing this without using recursion.

Example: 1 -> 3 -> 2 -> 5 -> 9
Your code should return 9 -> 5 -> 2 -> 3 -> 1
